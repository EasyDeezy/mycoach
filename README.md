# MyCoach

An AI-powered personal coaching assistant built with Claude.

## Setup

```bash
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
```

## Usage

```bash
python3 main.py
```

Commands during a session:
- `reset` — start a new conversation
- `quit` — exit

## Running Tests

```bash
pytest tests/
```
